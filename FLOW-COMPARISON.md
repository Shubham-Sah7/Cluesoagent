# Trupeer Creation Flows - Comparison

## Record Screen Flow (8 Screens)

### Purpose
Capture a new recording with full control over recording setup.

### Flow Steps
1. **Recording Mode Selection** - Choose screen/camera/audio setup
2. **Recording Setup** - Select source, preview camera/mic
3. **Recording State** - Floating controller during recording
4. **Processing** - AI analyzes the recording
5. **AI Understanding** - Shows insights and recommendations
6. **Choose Output** - Select desired output type
7. **Generating** - Creates selected output
8. **Review** - Preview and publish

### Time Estimate
- Setup: ~30 seconds
- Recording: Variable (user controlled)
- Processing: ~30 seconds
- Selection: ~15 seconds
- Generation: ~30 seconds
- **Total: ~2-5 minutes** (depending on recording length)

### Key Features
- Full recording control
- Camera/mic preview
- Pause/resume recording
- Real-time recording timer
- Floating controller

---

## Upload Recording Flow (7 Screens)

### Purpose
Transform an existing video into polished outputs quickly.

### Flow Steps
1. **Upload Modal** - Drag & drop or browse files
2. **Processing** - AI analyzes the uploaded video
3. **AI Understanding** - Shows insights and recommendations
4. **Choose Output** - Select desired output type
5. **Preview** - Smart recommendations before generating
6. **Generating** - Creates selected output
7. **Review** - Preview and publish

### Time Estimate
- Upload: ~10 seconds
- Processing: ~30 seconds
- Selection: ~15 seconds
- Preview: ~5 seconds
- Generation: ~30 seconds
- **Total: ~90 seconds**

### Key Features
- Drag & drop upload
- Recent uploads list
- Smart AI recommendations
- Preview with suggestions
- Faster than recording

## When to Use Each Flow

### Use Record Screen When:
- Creating new content from scratch
- Need to capture live demonstration
- Want to narrate while recording
- Need camera overlay
- Recording a walkthrough in real-time

### Use Upload Recording When:
- Already have a video file
- Recorded with external tool (Zoom, Loom, etc.)
- Want to repurpose existing content
- Need faster processing
- Have multiple videos to process

- Typography: Medium headings, regular body

## User Journey

### New User (First Time)
1. Lands on homepage
2. Sees "Record Screen" as primary action
3. Clicks and goes through 8-screen flow
4. Learns the product capabilities
5. Gets polished output

### Returning User (Has Videos)
1. Lands on homepage
2. Sees "Upload Recording" option
3. Clicks and goes through 7-screen flow
4. Faster path to value
5. Can process multiple videos quickly

### Power User
1. Uses both flows depending on need
2. Record Screen for new content
3. Upload Recording for existing videos
4. Appreciates the speed difference
5. Leverages smart recommendations

## Product Strategy

### Record Screen = Primary Value Prop
- Featured as primary action
- Larger, more prominent button
- Brand color background
- "Capture your screen with narration"

### Upload Recording = Secondary Path
- Still prominent but not primary
- Standard button styling
- "Transform an existing video"
- Faster alternative for existing content

### Start from Template = Tertiary Option
- Third option in quick actions
- For users who want structure first
- "Pick a proven structure"

This hierarchy guides users to the primary value proposition while offering alternatives for different use cases.
